# Определение типа документирующего комментария

## Неправильно

```bsl
// Параметры:
//  Параметры - Структура1 - некорректый тип
Процедура Неправильно(Параметры) Export
	// пустая
КонецПроцедуры

```

## Правильно

```bsl
// Параметры:
//  Параметры - Структура - корректный тип
Процедура Правильно(Параметры) Экспорт
	// пустая
КонецПроцедуры

```
