# Стандартные области структуры модуля

Проверяет регион обработчиков событий на методов относящихся только к обработчикам

## Неправильно

```bsl

#Область ОбработчикиСобытийФормы

Процедура Тест()
КонецПроцедуры

#КонецОбласти

```

## Правильно

```bsl

#Область ОбработчикиСобытийФормы

Процедура ПриОткрытии(Отмена)
КонецПроцедуры

#КонецОбласти

```
