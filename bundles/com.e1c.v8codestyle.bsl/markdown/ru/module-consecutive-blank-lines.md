# Проверка максимального количства пустых строк

## Неправильно

```bsl
Процедура ПередЗаписью(Отказ)


Если ОбменДанными.Загрузка Тогда
     Возврат;
КонецЕсли;



// код обработчика
// ...


КонецПроцедуры
```

## Правильно

```bsl
Процедура ПередЗаписью(Отказ)
Если ОбменДанными.Загрузка Тогда
     Возврат;
КонецЕсли;

// код обработчика
// ...
КонецПроцедуры
```
