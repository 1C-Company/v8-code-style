# Область пустая

Проверяет что область модуля пустая

## Неправильно

```bsl
#Область ОписаниеПеременных

// объявление переменных

#КонецОбласти

```

## Правильно

```bsl
#Область ОписаниеПеременных

Перем ВалютаУчета;
Перем АдресПоддержки;
...

#КонецОбласти
```
