# Аннотация написана канонически

Аннотации пишутся канонически (как в документации или Синтакс-помощнике).

## Неправильно

```bsl
&ИЗМЕНЕНИЕиконтроль("МояФункция")
Функция Расш1_МояФункция()
	
	#Удаление
	Возврат 1;
	#КонецУдаления
	
КонецФункции
```

## Правильно

```bsl
&ИзменениеИКонтроль("МояФункция")
Функция Расш1_МояФункция()
	
	#Удаление
	Возврат 1;
	#КонецУдаления
	
КонецФункции
```
