# Общий модуль должен иметь хотя бы один экспортный метод

Общий модуль должен иметь программный интерфейс.

## Неправильно

```bsl

Процедура Тест()
    //TODO
КонецПроцедуры

```

## Правильно

```bsl

#Область СлужебныйПрограммныйИнтерфейс 

Процедура Тест() Экспорт
    //TODO
КонецПроцедуры

#КонецОбласти

```
